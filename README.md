## Sobre o Projeto

Este aplicativo serve como um laboratório prático para testes, aprendizado e aperfeiçoamento de integrações web no Flutter. Atualmente, o app consome dados reais para exibir catálogos e categorias, focando não apenas no tráfego de dados (via JSON), mas também em entregar uma experiência de usuário (UX) elegante e fluida.

## Funcionalidades em Destaque

* **Consumo de APIs Externas:** Integração robusta utilizando requisições HTTP (GET).
* **Interface Dinâmica:** Renderização de Grids e Listas a partir do processamento de arquivos JSON (ex: Catálogo de Receitas).
* **Design:** Telas modernas utilizando `CustomScrollView`, efeitos *Parallax* com `SliverAppBar`, e componentes interativos do Material Design.
* **Resiliência:** Tratamento adequado de estados da tela (Carregamento, Erro de conexão e Dados Vazios).
* **Suporte a Temas:** Layout adaptável para Light e Dark Mode do sistema.

## Tecnologias Utilizadas

* **[Flutter](https://flutter.dev/):** Framework UI
* **[Dart](https://dart.dev/):** Linguagem de programação
* **[http](https://pub.dev/packages/http):** Pacote oficial para requisições de rede
